  "body": "## Gemini 요약 기능, 나만을 위한 맞춤 설정 가이드\n\nGemini는 텍스트를 빠르게 이해하고 핵심 내용을 파악하는 데 유용한 AI 도구입니다. 긴 문서나 웹 페이지를 요약하여 시간을 절약해주고, 아이디어를 빠르게 정리하는 데 도움을 줍니다. 또한, 다양한 글쓰기 스타일을 적용하여 초안 작성 시간을 단축하고, 창의적인 글쓰기를 지원합니다. Gemini의 요약 기능을 활용하면 정보 과부하에서 벗어나 중요한 내용에 집중할 수 있습니다.\n\n**Gemini 요약 기능은 다음과 같은 상황에서 유용합니다.**\n\n*   **뉴스 기사 요약:** 빠르게 주요 사건 파악\n*   **연구 논문 요약:** 핵심 아이디어 및 연구 결과 정리\n*   **회의록 요약:** 주요 결정 사항 및 액션 아이템 추출\n*   **이메일 요약:** 중요한 메시지 및 마감일 확인\n*   **소셜 미디어 게시물 요약:** 트렌드 및 주요 토론 주제 파악\n\n[주의] 메뉴/버튼 명칭은 업데이트로 달라질 수 있습니다.\n\n## 실습 준비\n요금: 무료 플랜 있음 / 유료 플랜 있음. 일부 고급 기능은 유료일 수 있음.\n실습 페이지: https://gemini.google.com/\n실습 목표: 기능 1회 실행 완료(설정/생성/저장까지)\n\n**실습 준비물**\n\n*   Google 계정 필요\n*   Gemini 페이지에 로그인 필요\n*   새 채팅 시작 또는 기존 채팅 활용\n*   요약할 텍스트 준비 (웹 페이지 URL 또는 텍스트 복사)\n\n## Gemini 요약 기능, 따라하며 익히기\n\n**1단계:** Gemini 페이지 접속 후, 새 채팅을 시작하거나 기존 채팅을 선택합니다. 화면 왼쪽 상단의 '+' 버튼을 클릭하여 새 채팅을 시작할 수 있습니다. 새 채팅 창이 나타나면 다음 단계를 진행합니다. 만약 기존 채팅을 활용하고 싶다면, 목록에서 원하는 채팅을 선택합니다. \n\n**2단계:** 요약하고 싶은 텍스트를 입력하거나 웹 페이지 URL을 입력합니다. 채팅 입력 창에 직접 텍스트를 붙여넣거나, \"[웹 페이지 주소] 요약해 줘.\" 와 같은 프롬프트를 입력합니다. 텍스트 입력 후, Gemini가 텍스트를 인식하는지 확인합니다. \n*실패 시 대안: 텍스트가 너무 길 경우, 짧게 나누어 시도합니다.\n\n**3단계:** 요약 요청 프롬프트를 입력합니다. \"요약해 줘.\", \"핵심 내용만 정리해 줘.\", \"5문장으로 요약해 줘.\" 와 같이 원하는 요약 방식을 구체적으로 지정합니다. 더 자세한 요약을 원하면 문장 수를 늘리거나, 특정 키워드를 포함하도록 요청할 수 있습니다. \n*실패 시 대안: 프롬프트의 표현을 바꿔서 다시 시도합니다. (예: \"핵심 내용만 요약해 줘\" -> \"가장 중요한 내용만 정리해 줘\")\n\n**4단계:** Gemini가 요약 결과를 표시하는지 확인합니다. 몇 초 후, Gemini는 입력된 텍스트 또는 URL에 대한 요약 결과를 채팅 창에 표시합니다. 요약 결과가 나타나지 않으면, 네트워크 연결 상태를 확인하거나 페이지를 새로고침합니다.\n*실패 시 대안: Gemini 페이지를 새로고침하거나, 잠시 후 다시 시도합니다.\n\n**5단계:** 요약 결과가 만족스럽지 않다면, 프롬프트를 수정하여 다시 시도합니다. 요약 결과가 너무 길거나 짧다면, 문장 수를 조절하거나, 특정 키워드를 포함하도록 프롬프트를 수정합니다. 예를 들어, \"[기존 요약 결과]에서 핵심 키워드 [키워드]를 포함하여 다시 요약해 줘.\" 와 같이 요청할 수 있습니다.\n*실패 시 대안: 원하는 키워드를 명확하게 지정하여 다시 요청합니다.\n\n**6단계:** 요약 결과를 복사하여 원하는 곳에 붙여넣습니다. 요약 결과 텍스트를 선택하고 복사(Ctrl+C)한 다음, 문서 편집기, 이메일, 메모장 등 원하는 곳에 붙여넣기(Ctrl+V)합니다.\n*실패 시 대안: 복사 단축키가 작동하지 않으면, 텍스트를 마우스 오른쪽 버튼으로 클릭하여 복사 메뉴를 선택합니다.\n\n**7단계:** 필요에 따라 요약 결과를 편집합니다. 복사된 요약 결과에서 불필요한 부분을 삭제하거나, 문장을 추가하여 내용을 보완합니다. 요약 결과는 사용 목적에 맞게 자유롭게 수정할 수 있습니다.\n*실패 시 대안: 편집 기능이 없는 텍스트 편집기 대신, 워드 프로세서(Microsoft Word, Google Docs)를 사용합니다.\n\n## 복사해서 바로 쓰는 Gemini 프롬프트 예시\n\n**프롬프트 1:**\n
