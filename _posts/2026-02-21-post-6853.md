## 구글 Lyria, AI 음악 생성 기능 공개: Vertex AI 기반 업데이트

2026년 2월 21일, 구글은 자사의 AI 음악 생성 모델 Lyria를 Vertex AI 플랫폼에 통합하여 공개했습니다. Lyria는 텍스트 프롬프트를 기반으로 새로운 악기 음악 트랙을 생성하는 기능을 제공하며, Vertex AI를 통해 더욱 다양한 사용자들이 접근하고 활용할 수 있게 되었습니다. **이번 업데이트는 음악 제작 분야에 AI 기술이 더욱 깊숙이 통합되는 중요한 전환점이 될 것으로 예상됩니다.**

## Lyria 주요 변경 사항

구글은 Lyria의 Vertex AI 통합을 통해 다음과 같은 주요 기능 업데이트를 제공합니다.

*   **텍스트 기반 음악 생성:** 사용자는 텍스트 프롬프트를 입력하여 원하는 스타일의 악기 음악 트랙을 생성할 수 있습니다.
*   **부정 프롬프트 지원:** 생성되는 음악에서 특정 요소를 제외하도록 모델을 안내하는 부정 프롬프트 기능을 사용할 수 있습니다.
*   **재현 가능한 결과:** 시드 값을 설정하여 동일한 프롬프트와 매개변수에 대해 일관된 음악 생성 결과를 얻을 수 있습니다.
*   **전 세계 지원:** Lyria 2는 전 세계에서 사용할 수 있으며, 데이터 저장 위치를 제어하기 위해 요청 시 특정 리전을 지정할 수 있습니다.
*   **SynthID 워터마킹:** 생성된 오디오에는 SynthID 워터마킹 기술이 적용되어, AI 생성 콘텐츠임을 식별할 수 있습니다.

이러한 변경 사항을 통해 사용자들은 더욱 쉽고 효율적으로 음악 제작에 AI 기술을 활용할 수 있게 되었습니다. 구체적인 사용법은 아래 '바로 확인/써보기' 섹션을 참조하십시오.

## Lyria 성능 및 제한 사항

Lyria는 텍스트 프롬프트를 기반으로 악기 음악 트랙을 생성하는 데 특화되어 있습니다. API 호출 빈도는 프로젝트별 분당 프롬프트 수에 따라 제한될 수 있으며, 이는 표준 Vertex AI Inference 한도에 따라 달라집니다. 오디오 클립 생성 시간은 일반적으로 10~20초 이내이지만, 사용량이 급증하는 경우 일부 요청이 대기열에 추가될 수 있습니다. 또한, 요청당 반환되는 오디오 클립 수는 지정된 `sample_count` 값에 따라 결정됩니다. 입력 프롬프트는 미국 영어(en-us)만 지원됩니다.

## 바로 확인/써보기

Lyria의 AI 음악 생성 기능을 직접 사용해 보려면 다음 단계를 따르세요.

1.  **Google Cloud 계정 로그인:** Google Cloud 콘솔에 로그인하거나, 계정이 없는 경우 새로 생성합니다. 신규 사용자는 무료 크레딧을 활용할 수 있습니다.
2.  **Vertex AI API 활성화:** Google Cloud 콘솔에서 Vertex AI API를 활성화합니다. API 활성화에는 Service Usage Admin IAM 역할이 필요합니다.
3.  **Vertex AI Studio 접속:** Google Cloud 콘솔에서 Vertex AI Studio > Media Studio 페이지로 이동합니다.
4.  **Lyria 모델 선택:** Media Studio에서 Lyria 모델 또는 음악 생성 옵션을 선택하고, 사용 가능한 경우 `lyria-002` 모델을 선택합니다.
5.  **프롬프트 입력 및 음악 생성:** 텍스트 프롬프트 작성 상자에 원하는 음악 스타일을 설명하는 텍스트를 입력하고, 필요에 따라 부정 프롬프트를 추가합니다. '생성' 버튼을 클릭하여 음악을 생성합니다.

## 요약 실행 절차

1.  Google Cloud 계정에 로그인합니다.
2.  Vertex AI API를 활성화합니다.
3.  Vertex AI Studio의 Media Studio 페이지로 이동합니다.
4.  Lyria 모델을 선택합니다.
5.  텍스트 프롬프트를 입력하고 음악을 생성합니다.

이 단계를 통해 Lyria의 AI 음악 생성 기능을 직접 경험하고, 다양한 음악 스타일을 탐색해 볼 수 있습니다.
